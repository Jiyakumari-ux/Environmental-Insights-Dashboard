# 🌦️ Environmental Insights Dashboard

An interactive Power BI dashboard built to monitor real-time weather conditions and air quality trends for different cities.

## 📌 Project Overview
This dashboard uses live weather and air quality data fetched from WeatherAPI and transforms it into meaningful visual insights.

The goal of this project is to make environmental data easier to understand through a clean and interactive dashboard.

## ✨ Features
- Real-time temperature updates
- Air Quality Index (AQI) monitoring
- Humidity analysis
- Wind speed tracking
- Pressure and visibility insights
- 7–10 day weather forecast
- City-wise comparison dashboard

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- WeatherAPI
- Data Visualization
